# Contapila

## Register
product

## Users & purpose
Personal double-entry bookkeepers who keep Beancount-style ledgers and want a fast, local, one-binary tool to check books and answer “what do I have / what did this month do?” — at a desk, with real numbers, not a marketing site.

Primary tasks: open a multi-ledger project, run check, scan balances / journal / P&L / net worth in a read-only browser UI.

## Personality
Quiet · Precise · Helix-defaults (opinionated, ready, no plugin circus)

## Anti-references
- Bank-app navy + gold “fintech luxury”
- SaaS hero dashboards with giant gradient metrics
- Fava clone that tries to match every pixel of Python Fava

## Design principles
1. Numbers first — tables and diagnostics beat decoration.
2. Trust via clarity — severity (error/warn/ok) is obvious; no magic.
3. Theme follows the room — light for day, dark for night desk; same structure.
4. Local tool density — familiar product chrome (nav + content), not a landing page.
